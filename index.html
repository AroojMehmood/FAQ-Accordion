Frequently Asked Questions

Common questions about web development

What is the difference between HTML and CSS?
HTML (HyperText Markup Language) is used to structure the content of a webpage, such as headings, paragraphs, and images. CSS (Cascading Style Sheets) is used to style that content — controlling colors, layout, fonts, and spacing.
What is responsive web design?
Responsive web design is an approach where a website automatically adjusts its layout and content to look good on different screen sizes, from mobile phones to tablets and desktops, usually using CSS media queries.
What is the difference between let, const, and var in JavaScript?
var is function-scoped and can be redeclared, which can lead to bugs. let is block-scoped and can be reassigned but not redeclared in the same scope. const is also block-scoped but cannot be reassigned after it is set, making it ideal for values that shouldn't change.
What is the DOM in JavaScript?
The DOM (Document Object Model) is a tree-like representation of a webpage that JavaScript can read and modify. It lets you select elements, change their content, update styles, and respond to user actions like clicks.
What is the difference between frontend and backend development?
Frontend development focuses on what users see and interact with in the browser, using HTML, CSS, and JavaScript. Backend development handles the server, database, and application logic that powers the frontend, often using languages like Node.js, Python, or PHP.
